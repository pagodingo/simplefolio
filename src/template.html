About me







My mission is to help organize, refresh, and execute your ideas.

My name is Daniel Archibald. Since 2018 I have had hands-on experience developing Web Applications. I can help you build scalable web apps that are capable of solving real world problems. If your team utilizes Github for version control, I'm very well educated on submitting pull requests to team projects. For front-end development, I design simple, clean and elegant UIs using Javascript, HTML & CSS, and frameworks such as React to produce responsive layouts. This portfolio and other projects I have produced in the past have been deployed using PaaS sites such as Heroku, Netlify and AWS. For back-end development PostgreSQL and MongoDB support my database management.
